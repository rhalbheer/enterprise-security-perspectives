# Enterprise Security Perspectives

Practical cybersecurity perspectives for boards, executives, and security leaders.

This repository contains long-form papers and role-based guides by [Roger Halbheer](https://www.linkedin.com/in/rhalbheer/) focused on enterprise security strategy, operating models, and execution.

## Suggested Reading Paths

- **Board / CEO / Business Leaders**: Start with [Your Security Model Is Too Slow to Matter](Security%20at%20AI%20Speed/01%20-%20Foundation%20%28START%20HERE%29/Your%20Security%20Model%20Is%20Too%20Slow%20to%20Matter.md), then read your role guide in [Security at AI Speed/04 - Role Guides](Security%20at%20AI%20Speed/04%20-%20Role%20Guides/)
- **CISO / CIO / CTO**: Read the AI Speed foundation, then continue with [02 - Operating Model](Security%20at%20AI%20Speed/02%20-%20Operating%20Model/) and [03 - Execution](Security%20at%20AI%20Speed/03%20-%20Execution/)
- **Compliance / Audit / DPO**: Read [Threat-Led Governance](Security%20at%20AI%20Speed/02%20-%20Operating%20Model/Threat-Led%20Governance.md), then role-specific guides
